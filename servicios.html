PLATAFORMAS LABS MOBILE

Generamos soluciones enfocadas en las necesidades de cada segmento de clientes

  • Labs Mobile Web

    Plataforma web disponible 7x24 que permite el envío de campañas masivas vía SMS. Permite la programación de campañas unidireccionales y bidireccionales, donde podrás obtener reportes en línea.

  • Labs Mobile API

    Nuestro servicio permite la integración vía API o WSD con las plataformas del cliente. Contamos con múltiples métodos de integración, ya sea SOAP, FTP, Rest o cualquiera que el cliente necesite.
